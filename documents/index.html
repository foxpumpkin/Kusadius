開発環境および使用したライブラリ

OS - Linux Mint 17 Qiana, Mac OS X yosemite
当初研究用のLinux環境で開発していたが，Windowsへの移植作業に失敗しカーネルが破壊された．
復旧の見込みが立たなかったので，予備の環境として構築済みだったMac OSに移行した．

H/W - CPU(Intel Core i5 2.6 GHz) RAM(8GB)
Linuxはパーティションを切ってネイティブにインストールしていたので，ハードウェアは同じである．

ライブラリ - SDL(net,ttf,image) boost(1.55)
当初の構想として，Windows，Mac，Linuxのクロス環境での通信対戦が目標だったので Simple DirectMedia Layerを採用した．
他のエンジンとしてUnity，Unrealが候補であったがLinuxで利用できないことから不採用となった．
boostは当初まったく使っていなかった（スレッドにはC++11の標準ライブラリを使っていた）が，Windowsに対応しているgccが古くまた32bit版であることからコンパイルできず， boost::thread，boost::mutexを試したさいのなごり．

ホストの指定と通信方式

通信ホストは実行フィアルと同じディレクトリにあるconnection.iniの一行目のみを読む．
読み込めないばあいはlocalhostに設定される．
インターネットレイヤはIPv4でのみ動作検証している．
トランスポートレイヤのプロトコルはTCPを採用した．即時性の必要な通信であればUDPが基本であるが， Kusadiusは直接撃ちあうゲームでないため信頼性を重視した採用となった．

必要なパッケージ

boost, SDL2, SDL2_net, SDL2_image, SDL2_ttfが必要である．
Ubuntuではapt-get install libboost1.55-all-dev libsdl2-dev libsdl2-image-dev libsdl2-ttf-dev libsdl2-net-devを管理者権限で実行すればよい．
Mac OSではhomebrewなどで必要なライブラリを取得されたい．
コンパイルにはgcc4.9以上が必要である．なお，makefileにはg++-4.9のコマンド名で記述されている．環境によっては編集が必要である．
